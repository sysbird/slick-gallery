== Description ==
Mobile touch slider with recent posts featured-image.

= Features =
* The plugin displays a slider by shortcode.  
* The slider displays recent posts featured-image.  
* Images on the slider link to the post page.  
* It's compatible with responsive web design and mobile touch.   
* [Swiper](http://www.idangero.us/sliders/swiper/) the jQuery plugin is licensed under GPL & MIT.  

= Usage =
1. Set featured-image in post.  
2. Please write shortcode [featured-images-swiper] in the template.  
3. You can set number of thumbnail the slider.  
   example. [featured-images-swiper number="10"]  
   number: umber of thumbnail the slider(default:10)  

[Demo](http://www.sysbird.jp/birdsite/2007/09/21/20070921-hakodate/)

1. Upload the "Featured Images Swiper" folder to the plugins directory in your WordPress installation.  
2. Go to plugins list and activate "Featured Images Swiper".
